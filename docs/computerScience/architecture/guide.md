# 基础平台篇

## 应用程序的基础架构

- 中央处理器 + 存储 + 一系列的输入输出设备 就是冯·诺依曼计算机体系

### 为何电脑能够完成这么多复杂而多样化的工作?

1. 第一是可编程性。
2. 第二开放的外部设备支持
3. 电脑的 CPU 是一个非常简洁的模型，它只读入和写出数据，对数据进行 计算

### 进化史

- 编程语言 + 编译器 =》操作系统 解决软件治理和基础编程接口问题
