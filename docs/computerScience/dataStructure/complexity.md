# 复杂度

## 分类
* 空间复杂度和时间复杂度

### 时间复杂度

* 时间复杂度是描述算法运行的时间

### 常见的时间复杂度

| 名称 |  运行时间 T(n)   | 时间举例 |   算法举例   |
| :--: | :------: | :--------------: | :------------: |
| 常数 |       O(1)      |    3   |      -      |
| 线性 | On） | n | 操作数组 |
| 平方 | O(n^2) | 双循环 | 冒泡排序 |
| 对数 | O(log(n)) | 对数 | 二分查找 |

### 空间复杂度

* 空间复杂度是对算法运行过程中临时占用空间大小的度量

* 常见的复杂度不多，从低到高排列就这么几个: O(1) 、 O(log(n)) 、 O(n) 、 O(n2) ，等学完 后面的章节你会发现，复杂度基本上逃不走，都是上面这个几个。

### 分类

* 最好复杂度  最坏复杂度  平均复杂度
